= Техническая поддержка

Компания "ДоксВижн" предоставляет сервис технической поддержки, уровень которого зависит от приобретённой редакции лицензии. Вы можете получить всю необходимую информацию на http://support.{dv}.com[нашем сайте].

Дополнительно компаниями-партнерами по внедрению могут оказываться услуги сопровождения внедренного решения. Если вы эксплуатируете систему, то свяжитесь с вашим поставщиком для уточнения деталей.
